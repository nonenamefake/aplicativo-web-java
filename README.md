Aplicativo web Java  
Aplicativo web java de compras (sin Apis) con uso de maven spring boot jpa hibernate mysql  
Publicacion de mi proyecto con fines de visualizacion y de retroalimentacion del publico con mayor conocimiento en el ambito de la programacion web (BACKEND)  

Intrucciones para el funcionamiento de las paginas de administracion despues de tener preparado el mysql:  
Con el uso de una IDE java, se recomienda Visual Studio code, explora el apartado de segurity  
si usa Visual Studio code, leer los Requisistos.txt del proyecto   
dentro del archivo security.java, agregar las direcciones /empleado/** como permitidos  
dentro del archivo securityemp.java, eliminar la restriccion del /empleado/**  
dentro de la pagina web usando el puerto asignado por spring boot, crear un empleado  
restablecer las modificaciones de los archivos de security
